Это проект разработал Михаил Чумаченко.

Учителя: Руслан Прокарпчук и Андрей Чаховский
Локация: компания Lohika


Проект представляет собой приложения для определения погоды по координатам.
Изначально есть 7 мест из которых можно выбрать и узнать погоду. Также можно ввести свои координаты и узнать о погоде в любом уголке мира.
Информацию, которую можно узнать:
1) Температура
2) Облачность
3) Скорость ветра
4) Влажность
5) Атмосферное давление
