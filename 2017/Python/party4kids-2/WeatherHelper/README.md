Это проект разработал Константин Федосеев.

Учителя: Руслан Прокарпчук и Андрей Чаховский
Локация: компания Lohika


Проект представляет собой реализацию настольной игры "Novem". Игра предназначена для игры вдвоём.
Вот правила к оригинальной игре(они применимы и к этой):

На игровой доске установлены пластинки с цифрами в трех горизонтальных и вертикальных рядах;
Один из игроков играет используя горизонтальные ряды, другой вертикальные. 
Оба выбирают один из рядов, не сообщая о выборе другому игроку. 
Далее игрок-нападающий берет себе пластинку из той клетки, где пересекаются значения выбранных вертикального и горизонтального рядов. 
После этого очередь нападения переходит к другому игроку, и так далее.
Игру выигрывает тот игрок, который из пластинок соберет большее количество очков.
