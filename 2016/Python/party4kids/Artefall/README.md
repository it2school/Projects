Это проект разработал Артём Максимов.

Учителя: Игорь Козыренко и Вадим Васильев.
Локация: компания Ciklum


Программа представляет собой текстовую RPG. Этот жанр игры был популярным на заре компьютерных игр.
В игре нет никакого графического отображения, есть только текстовые представления события и действия. 
Это пока только начало игры. Альфа-версия
