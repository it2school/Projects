Это проект разработал Михаил Чумаченко

Учителя: Николай Гуменюк и Никита Добрянский
Локация: коворкинг Терминал42


Проект представляет собой шифрование слов с помощью шифра Массонов. 
Зашифрованный текст рисуется символами и отображается на экране.
Шифровать можно только текст введенный на английском языке.
В файле "cipher.png" можно посмотреть, как шифруется текст.
