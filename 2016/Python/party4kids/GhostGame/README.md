Это проект разработал Артём Карандеев

Учитель: Богдан Свердлов
Локация: Библиотека им А.П. Гайдара, г. Одесса, Украина


Программа представляет собой игру, в которой нужно убежать от привидения. 
Вы передвигаетесь между комнатами и всегда есть несколько дверей куда можно пойти дальше, но берегитесь, за одной из них привидение! 
Чем больше раз вы сможете избежать привидения, тем выше будет ваш счёт.
